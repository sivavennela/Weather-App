# Weather App

This is a Django-based Weather App that allows users to search for real-time weather information by entering a city name.
It fetches live data from the OpenWeather API and displays it in a clean UI with dynamic backgrounds based on weather conditions.

## âœ¨ Features 

### Backend

- Fetches real-time weather data using OpenWeather API
- Handles user input and API responses using Django views
- Error handling for invalid city names or API failures
- Passes weather data dynamically to templates

### Frontend

- Clean and responsive user interface
- Inline city search form with instant results
- Dynamic background changes based on weather (sunny, rainy, cloudy)
- Displays temperature, humidity, pressure, and weather conditions
